# Flow of EPTT Fluid in an Axisymmetric pipe with Wall Slip - CFD Model

This repository contains the Computational Fluid Dynamics (CFD) model files for simulating the pipe flow of a Phan-Thien-Tanner (PTT) fluid in an axisymmetric pipe with wall slip.
